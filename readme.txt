Use the Emailmanager API to send transactional emails from your blog.

== Description ==

Replaces Wordpress' wp_mail() function to send mail through Emailmanager's API


Note: you need a Emailmanager account set up to use this plugin.

Limitations:

*   Email attachements are not supported
*   Additional headers sent to the wp_mail function are not supported
*   Only text based emails are supported

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Extract and upload 'emailmanager' directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
